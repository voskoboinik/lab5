## Какие интерактивности добавлены на страницу?
1. Кнопка «Наверх» — появляется при прокрутке и возвращает страницу вверх.  
2. Аккордеон — разворачивает и сворачивает текстовые блоки.  
3. Фотогалерея с кнопками-фильтрами (все, учёба, хобби и т.д.).  
4. Модальное окно с увеличением фото при клике.

## Где использовался ИИ, а где код писался вручную?
- Код для кнопки «Наверх» и галереи с фильтрацией — создан с помощью ИИ.  
- Код аккордеона и модального окна — адаптирован вручную.  
- Стили кнопок и позиционирование элементов — написаны вручную.

## Что показалось самым интересным или сложным?
Самым интересным было добавлять интерактивность и видеть результат в браузере.  
Сложнее всего оказалось сделать, чтобы аккордеон корректно закрывал остальные блоки при открытии нового.

## Какие запросы делали LLM?
1. «Создай кнопку “Наверх” на JavaScript, которая появляется при прокрутке вниз.»  
2. «Напиши аккордеон на чистом JS, чтобы открывался один блок за раз.»  
3. «Сделай фильтрацию фотогалереи по категориям.»  
4. «Добавь модальное окно для увеличения фото при клике.»
